# Ecoinvest UAE - Sitio Web Moderno

Un sitio web profesional y completamente responsivo para una plataforma de inversión sostenible en los Emiratos Árabes Unidos.

## 🌟 Características Principales

- **Diseño Responsivo:** Se adapta perfectamente a cualquier dispositivo
- **Moderno y Profesional:** Interfaz limpia y atractiva
- **Optimizado para Rendimiento:** Carga rápida y fluida
- **Menú Interactivo:** Hamburguesa adaptativa en móviles
- **Animaciones Suaves:** Transiciones y efectos visuales
- **Formulario de Contacto:** Completamente funcional
- **SEO Friendly:** Optimizado para motores de búsqueda

## 🚀 Inicio Rápido

### Con Python
```bash
cd automatic-chainsaw
python -m http.server 8000
# Accede a http://localhost:8000
```

### Con Node.js
```bash
npm install
npm start
# Accede a http://localhost:8080
```

### Directamente
Solo abre `index.html` en tu navegador favorito.

## 🎨 Secciones del Sitio

1. **Navegación** - Header sticky con menú responsivo
2. **Hero** - Sección principal llamativa
3. **Servicios** - 4 servicios principales (Energía, Sostenibilidad, Fondos, ESG)
4. **Sobre Nosotros** - Info de la empresa + estadísticas
5. **Proyectos** - 3 proyectos destacados con ROI
6. **Contacto** - Formulario + información de contacto
7. **Footer** - Enlaces legales y rápidos

## 🎯 Personalizaciones Comunes

### Cambiar logo/nombre
En `index.html`, busca `<div class="logo">ECOINVEST</div>`

### Actualizar colores
En `css/styles.css`, modifica las variables en `:root`

### Agregar contenido
Edita directamente en `index.html` las secciones que necesites

### Cambiar información de contacto
Busca `info@ecoinvest.ae` y reemplaza con tu email
